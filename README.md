PRACTICE QUIZ QUESTION
Write a program for a juice shop. Read N juices sold and price of each. If price < 100 tag “Small Juice”. Sum total and count. Use IO, variables, loops and if/else. 
MAIN QUIZ#01 QUESTION
Write a program for a school transport service. Read N students and monthly bus fee for each. If fee < 900 print “Concession Given”. Display the total of all fees and the number of students with concession. Use variables, input/output, loops and if/else.
